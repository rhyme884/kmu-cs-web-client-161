  2. Describe difference between Markup language(HTML) and Programming language, such as Java (10 point)

  3. Describe what do the following tags mean(4 point)
    • pre: 입력한 그 내용 그대로 표현해준다.
    • p: 문단을 정의해 준다. 'p'가 닫힌 후 그 밑 한 줄을 띄워준다.
    • br: 개행

  4. Describe diffrence between using strong, em tag and b, i tag (5 point). Answer: (strong,em) 같은 경우는 강조를 위해 사용하는 것이고, (b,i)는 단순히 글자의 외향에 변화를 주기 위해 사용한다.
  5. In the img tag, describe what alt tag means and how it differs from title attribute(5 point). Answer: alt : a message that will be shown when the image is not available,
    title : a message shown when a pointer is over the image
  6. In table, describe difference between th and td (5 point). Answer: td는 평범한 cell을 가리키지만, th(table heading)는 cell안의 text가 'b'와 text:align:center가 된 형태를 가진다.
